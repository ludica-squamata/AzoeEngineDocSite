Es un mundo gris e inhóspito. La naturaleza perece, las plantas y animales mueren, y la tierra se ha convertido en un páramo desolado. Extrañas criaturas mágicas vagan a su antojo por doquier; algunas son inteligentes, otras simplemente salvajes. Y se dice que ni siquiera los que mueren pueden descansar en paz.

En tu camino harás muchos amigos, y también ganarás muchos enemigos. Lucharás contra poderosos monstruos mágicos y tu aventura no conocerá barreras. En tus manos hallarás el poder para cambiar el destino del mundo, o podrás condenarlo a la destrucción absoluta.

El mundo te necesita, y quizás sientas que debes acudir a su rescate… o tal vez quieras tomar provecho de esa necesidad. Quizás el mundo no te importe tanto, porque estés tras una cosa en particular. Sea como fuere, ésta es la tarea, héroe, y eres tú quien ha de enfrentarla.

Proyecto Mano's-Gifts

El proyecto Mano's-gifts propone un juego de rol (RPG) donde tú, el héroe, tendrás que restaurar el equilibrio de la existencia del mundo, devolviendo a la naturaleza a su estado original, y destruyendo (o domando) a la salvaje magia que da forma a los más retorcidos monstruos que habitan la Tierra.

Podrás:

  • Crear a tu personaje, el héroe, a medida
  • Explorar dos peligrosos mundos, y hacer aliados y enemigos por doquier
  • Influir activamente en la historia con cada decisión que tomes

Instrucciones para probar el Engine

Para hacer una prueba del engine como está hecho hasta ahora, deberás seguir los siguientes pasos:

  1. Descarga Python 3.4.4. Recuerda bajar la versión correcta para tu sistema operativo (32 o 64 bits)
  2. Instala Python 3.4.4
  3. Descarga el build especial de Pygame 1.9.2a0 para Python 3.4
    • Si tu sistema es de 32bit, deberás bajar el build pygame-1.9.2a0-cp34-none-win32.whl
    • Si tu sistema es de 64bit, deberás bajar el build pygame-1.9.2a0-cp34-none-win_amd64.whl

  4. Notarás que el instalador de pygame es un archivo whl. Windows no reconoce este tipo de achivos por defecto, por lo que no puedes instalarlo con solo un doble click.

    Para instalar pygame, sigue estos pasos

    1. Abre el "Simbolo de Sistema" (CMD) como Administrador
    2. Usando el comando cd (explicación), ve hacia la ruta donde esté el archivo whl
    3. Introduce el siguiente comando:

      pip install some-package.whl

      (some-package.whl es el archivo whl de pygame, asegúrate de escribir correctamente el nombre del arhivo)

  5. Una vez que tengas python y pygame instalado, puedes bajar el zip del engine.
  6. Ahora solo queda abrir el archivo main.py que se encuentra donde hayas descomprimido el arhivo.
