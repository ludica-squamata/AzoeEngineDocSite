Notas Técnicas

Diálogos

  • Se estructuran como árboles de diálogo.
  • El jugador puede elegir exáctamente lo que el personaje va a decir.
  • Cada diálogo es un archivo separado.
  • El carisma (o cualquier otro atributo) puede determinar si algunas opciones de diálogo se mostrarán si la puntuación del atributo es mayor al requisito de dicha opción.
  • Fragmentos de los diálogos pueden resaltarse mediante tags, estilo <tag>palabra</tag>.
  • Los bloques de texto grandes pueden ser desplazados hacia arriba y abajo, incluso si son opciones a elegir de un menú circular.
  • Las puntuaciones de característica del héroe ahora también habilitan a mostrar (o no) partes del diálogo de los NPCs. Lo mismo sucede con los objetos que el héroe tenga encima.
  • Ciertas opciones dentro del árbol pueden cambiar puntuaciones de un personaje mediante eventos. Mediante esta mecánica, también podrían setear event flags.
  • Si es el player el que inicia el diálogo, se abre un menú circular que le permite elegir de qué quiere hablar. Si, mediante AI, es el NPC el que inicia el diálogo, este menú no aparece.
  • El player puede elegir lo que va a decir mediante un menú circular. Esto permite que las opciones de diálogo del héroe sean más extensas que una línea.
  • Al tocar “acción” sobre un elemento del escenario, se abre una descripción de 1 bloque de texto. Este bloque desaparece al tocar nuevamente acción o cancelar diálogo.

Reloj Multifuncional

El engine soporta dos clases de Relojes para el mantenimiento del tiempo en el juego. El reloj "ficticio" corre con los ticks del juego, y puede ser comprimido o dilatado con respecto al tiempo del mundo real como se disponga. El reloj Real, por otro lado, está atado al tiempo del sistema y cambia cuando cambia el tiempo del mundo real.

Tags Condicionales

Cualquier Tag puede escribirse con requisitos que se deben cumplir para que ésta se muestre. El engine puede discriminar por los atributos (altos y bajos) de los diversos hablantes ("interlocutores") del diálogo para determinar si la tag en cuestión se muestra o no.
